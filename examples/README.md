# Examples

This directory contains optional, repo-only examples.

They are **not** part of the published SlopSentinel package and are not included
in `pytest` runs by default.

Contents:

- `plugin-openai-slop/`: an example rule plugin package
- `plugin-security/`: an example security-oriented rule plugin package
- `custom-reporter/`: an example script that renders a JSON scan report into a
  custom format
